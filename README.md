# Coffee Shop Website

This project is a static coffee shop website developed using HTML and CSS.
It presents a clean layout with sections for navigation, introduction, shop details, menu items, and footer information.

Project Description

The website is designed to represent a coffee shop’s online presence.
It focuses on layout design, color usage, responsiveness, and basic user interface structure using only front-end technologies.

Features

Header with navigation menu

Hero section with background image and call-to-action button

About and specialties section

Menu section displaying coffee, tea, and pastries

Responsive design for smaller screen sizes

Footer with copyright information

Technologies Used

HTML5

CSS3

Google Fonts (Poppins)

Project Structure

index.html – Main HTML file containing structure and styling

All CSS is written internally within the HTML file.

How to Run the Project

Download or copy the project file

Save it as index.html

Open the file in any modern web browser

Notes

This is a front-end only project

No backend or database is used

Content and styles can be customized easily
